Bella Pele Bot é um chatbot de atendimento e agendamento para clínicas estéticas e dermatológicas, integrado ao WhatsApp Business API e com inteligência artificial real via Claude (Anthropic).
A assistente virtual Sofia conversa com os pacientes de forma natural, apresenta procedimentos e preços, coleta os dados do agendamento e notifica a clínica automaticamente no WhatsApp assim que uma consulta é marcada. Tudo isso sem nenhuma intervenção humana.
A clínica acompanha todos os agendamentos por um painel admin web, acessível de qualquer dispositivo, sem precisar instalar nada.
